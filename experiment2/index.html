Experiment 2 - A Procedurally Generated Living Impression

Description

This project offers a generative, living impression of this inspiring image:

Technical

Relation to inspring image:

  • Color: The sky, grass, clouds, and flower colors were sampled from the inspiring image using my eyes, I eyeballed it using a hex color picker.
  • Primitives: The clouds and flowers are rendered with elipses and circles.
  • Polygons: The bushes are randomly positioned and generated using randomly shaped polygons that change based on the seed
  • Composition: Different random seed values lead to differently shaped bushes and differently positioned clouds/flowers
  • Life: The position of clouds is animated over time as if to simulate them drifting off with the wind. The user can also spawn more flowers/clouds by clicking around the landscape

Reflection

As I went through the experimentation process for Experiment 2, I learned a lot about how to generate polygons and randomly create them based on a seed
